🐞 Bug Report

Login Button Unresponsive

Bug ID BR-001
Application Demo Web Application
Module Authentication
Severity High
Priority High
Status Open
Environment Windows 11 • Chrome Latest

Summary

The Login button does not respond after entering valid credentials, preventing users from accessing their accounts.

Preconditions

  • User account already exists.
  • Stable internet connection.

Steps to Reproduce

  1. Open the Login page.
  2. Enter a valid email address.
  3. Enter the correct password.
  4. Click the Login button.

Expected Result

User should successfully log in and be redirected to the dashboard.

Actual Result

Clicking the Login button produces no response and the user remains on the Login page.
